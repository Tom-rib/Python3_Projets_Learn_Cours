# 📚 Data Science with Python – Cours & Projets

Un cours complet de Python pour débutants, idéal pour apprendre la programmation et les fondamentaux en data science.

---

## 🎯 Contenu

### 📖 **Cours Python** (`Python_Cours/`)

**31 chapitres progressifs** couvrant :

- **Bases** : variables, types, opérateurs, arithmétique (chap. 0-6)
- **Strings & Booléens** : manipulation de texte, logique booléenne (chap. 7-9)
- **Contrôle de flux** : conditions, boucles (chap. 12-13)
- **Fonctions & Modularité** : définition, paramètres, retours (chap. 14-15)
- **Structures de données** : listes, tuples, dictionnaires, sets (chap. 16-18)
- **Fichiers & Modules** : E/S, imports (chap. 22, 28)
- **Outils avancés** : itérables, compréhensions, slices (chap. 23, 31)
- **Bonus** : dates/heures, tests unitaires, pandas (chap. 30, 29, xx)

Chaque chapitre inclut **explications + exemples exécutables**.

---

### 💻 **Mini-Projets** (`Mini_Projet/`)

1. **🎮 Pendu** – Jeu à deux joueurs (le Hangman)
   - Joueur 1 pense à un mot, Joueur 2 le devine
   - 6 erreurs max avant défaite
   - Fichier : `Mini_Projet/Hangerman/hangerman.py`

2. **🏗️ Pyramide** – Génération de pyramides ASCII
   - Crée des pyramides de tailles variables
   - Toit + porte paramétrable
   - Fichier : `Mini_Projet/Pyramide/pyramide.py`

3. **🔴 Puissance 4** – Jeu classique
   - Jeu à deux joueurs
   - Grille 7x5, 6 erreurs max
   - Fichier : `Mini_Projet/puissance4/puissance4.py`

---

### 🏋️ **Exercices** (`Python_Cours/exos/`)

- **Échauffement J02** : 10 micro-exercices pour vérifier les bases (30 min)
- **Exercices progressifs** : débutant → intermédiaire → avancé
- **Énoncés** + **Solutions corrigées** disponibles
- **Topics** : arithmétique, strings, boucles, listes, fonctions, dictionnaires, etc.

## 🚀 Comment utiliser

### Commencer le cours

```bash
# Lire un chapitre théorique
python Python_Cours/chap_01_ce_tuto.py

# Faire les exercices d'échauffement
python Python_Cours/exos/echauffement_j02.py

# Tester les exercices d'un chapitre
python Python_Cours/exos/exos_12_if_then.py
```

### Lancer un mini-projet

```bash
# Jouer au Pendu
python Mini_Projet/Hangerman/hangerman.py

# Générer une pyramide
python Mini_Projet/Pyramide/pyramide.py

# Jouer au Puissance 4
python Mini_Projet/puissance4/puissance4.py
```

### Structure recommandée

1. **Lisez** le chapitre théorique
2. **Testez** les exemples dans un interpréteur
3. **Faites** les exercices du chapitre
4. **Progressez** au chapitre suivant
5. **Consolidez** avec les mini-projets

## 💡 Conseils d'apprentissage

✅ **À faire** :
- Tapez le code au lieu de le copier-coller
- Testez les exemples dans un interpréteur
- Modifiez les exemples pour explorer
- Faites les exercices avant de regarder les solutions
- Pratiquez régulièrement (un peu chaque jour)

❌ **À éviter** :
- Lire sans pratiquer
- Sauter les exercices
- Utiliser ChatGPT avant d'essayer soi-même
- Passer au chapitre suivant sans maîtriser le courant
